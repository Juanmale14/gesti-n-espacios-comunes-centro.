
# Gestión de Espacios Comunes

## Descripción
Esta aplicación web permite la **reserva y administración de espacios educativos** de forma sencilla e intuitiva. Está diseñada para facilitar la gestión de espacios comunes como la biblioteca, el salón de actos, el aula de informática y el gimnasio.

## Características
- Selección visual de espacios mediante tarjetas interactivas.
- Registro de docentes participantes con selección de departamentos.
- Configuración de horarios semanales por espacio.
- Exportación de la reserva en formato PDF y Excel.
- Interfaz amigable y organizada por secciones.

## Estructura de la Aplicación
- **Selección de Espacios**: Permite elegir uno o varios espacios disponibles.
- **Detalles de la Reserva**:
  - Docentes participantes (hasta 5).
  - Fecha de la reserva.
  - Horarios por espacio.
- **Resumen y Exportación**: Muestra un resumen de la reserva y permite exportar los datos.

## Cómo usar
1. Abre `index.html` en tu navegador.
2. Selecciona los espacios que deseas reservar.
3. Completa los datos de los docentes y la fecha.
4. Guarda la configuración.
5. Exporta el resumen en el formato deseado.

## Requisitos
Este proyecto no requiere instalación de dependencias externas. Solo necesitas un navegador web moderno.
